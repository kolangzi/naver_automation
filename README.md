# 네이버 서로이웃 자동 신청 도구

네이버 블로그 서로이웃 자동 신청 + AI 댓글 작성 GUI 도구

## 주요 기능

### 기능 1: 서로이웃 신청
공감 계정 중 이웃추가 가능한 계정에 자동으로 서로이웃 신청하고, 신청 성공한 계정의 최신글에 AI 댓글을 남깁니다.

### 기능 2: 서로이웃 댓글
내 서로이웃 관리 페이지에서 이웃을 수집하여, 기준 날짜 이후 글을 올린 이웃의 최신글에 AI 댓글을 자동으로 남깁니다.

## 요구사항

- macOS 10.15 (Catalina) 이상
- Python 3.8 이상
- 인터넷 연결
- Gemini API 키 (댓글 기능 사용 시 필수)

## 설치 및 실행

```bash
# 1. 설치 (최초 1회)
./install.sh

# 2. 실행
./run.sh
```

### 수동 설치 (install.sh가 안 될 경우)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
playwright install chromium
python main.py
```

## 사용법

### 기능 1: 서로이웃 신청

1. **블로그 URL**: 공감 계정을 추출할 네이버 블로그 글 주소 입력
   - 예: `https://blog.naver.com/username/123456789`
2. **네이버 ID**: 로그인할 네이버 아이디 입력 (미입력 시 기본값 사용)
3. **비밀번호**: 비밀번호 입력
4. **서로이웃 메시지**: 서로이웃 신청 시 전달할 메시지 (커스텀 가능)
5. **댓글 기능** (토글 ON/OFF):
   - **Gemini API 키 필수** — AI가 글 내용을 읽고 맥락에 맞는 댓글 자동 생성
   - 30대 여성 말투, 이모티콘 포함, 15자 내외
   - AI 생성 실패 시 3회 재시도 후 작업 종료
6. **[시작]** 버튼 클릭

### 기능 2: 서로이웃 댓글

1. **네이버 ID / 비밀번호**: 로그인 정보 입력
2. **Gemini API 키**: AI 댓글 생성에 필수
3. **그룹 이름**: 서로이웃 그룹 이름 (기본값: 이웃1)
4. **기준 날짜**: 이 날짜 이후 글을 올린 이웃만 대상 (기본값: 오늘)
5. **[시작]** 버튼 클릭

## 동작 방식

### 기능 1: 서로이웃 신청

1. 네이버 로그인 (캡차 발생 시 30초 내 수동 로그인)
2. 블로그 글에 공감(하트) 클릭 (이미 공감한 경우 스킵)
3. 공감 히스토리 페이지에서 이웃추가 가능한 계정 추출 (페이지네이션 자동 처리)
4. 각 계정에 서로이웃 신청 — 실제 성공 100명까지 (스킵된 계정 제외)
5. 신청 성공한 계정의 최신 글에 AI 댓글 작성

### 기능 2: 서로이웃 댓글

1. 네이버 로그인
2. 서로이웃 관리 페이지 접속
3. 지정한 그룹 선택 → 업데이트순 정렬
4. 기준 날짜 이후 글을 올린 이웃 목록 수집 (조기 종료)
5. 각 이웃의 최신 글 방문
6. 이미 내 댓글이 있으면 스킵
7. AI로 글 맥락에 맞는 댓글 생성 → 등록

## Gemini API 키 발급

1. https://aistudio.google.com/apikey 접속
2. Google 계정 로그인
3. "Create API Key" 클릭
4. 발급된 키를 프로그램의 "Gemini API 키" 필드에 입력
