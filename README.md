# 네이버 서로이웃 매니저

네이버 블로그 서로이웃 자동 신청 + AI 댓글/대댓글 작성 GUI 도구

## 주요 기능

| 기능 | 설명 |
|------|------|
| **서로이웃 신청** | 공감 계정 중 이웃추가 가능한 계정에 자동 서로이웃 신청 |
| **서로이웃 댓글** | 서로이웃의 최신글에 공감 + AI 댓글 자동 작성 |
| **대댓글** | 내 블로그 글의 댓글에 AI 대댓글 자동 작성 |

## 요구사항

| 항목 | 조건 |
|------|------|
| OS | macOS 10.15 (Catalina) 이상 |
| Python | 3.8 이상 |
| 네트워크 | 인터넷 연결 필수 |
| API 키 | Gemini API 키 (기능 2, 3 사용 시 필수) |

## 사용법

> 모든 기능 공통: **[시작]** 클릭 후 브라우저에서 **비밀번호 입력 → 로그인** (3분 이내)

### 기능 1: 서로이웃 신청

| 입력 항목 | 설명 | 기본값 |
|-----------|------|--------|
| 블로그 URL | 공감 계정을 추출할 블로그 글 주소 | — |
| 네이버 ID | 로그인에 사용할 네이버 아이디 | — |
| 서로이웃 메시지 | 신청 시 전달할 메시지 | 커스텀 가능 |

> 블로그 URL 예시: `https://blog.naver.com/username/123456789`

### 기능 2: 서로이웃 댓글

| 입력 항목 | 설명 | 기본값 |
|-----------|------|--------|
| 네이버 ID | 로그인에 사용할 네이버 아이디 | — |
| Gemini API 키 | AI 댓글 생성용 API 키 | — |
| 그룹 이름 | 서로이웃 그룹 이름 | 이웃1 |
| 정렬 기준 | 업데이트순 / 이웃추가순 | 업데이트순 |

> - _업데이트순_ — 최근 글 날짜로 필터링
> - _이웃추가순_ — 이웃추가일로 필터링

### 기능 3: 대댓글

| 입력 항목 | 설명 | 기본값 |

| 네이버 ID | 로그인에 사용할 네이버 아이디 | — |

## 동작 방식

### 기능 1: 서로이웃 신청

```
로그인 → 블로그 글 공감 클릭 → 공감 히스토리에서 계정 추출 → 서로이웃 신청
```

1. 네이버 로그인 (아이디 자동 입력 → 비밀번호 수동 입력)
2. 블로그 글에 공감 클릭 (이미 공감한 경우 스킵)
3. 공감 히스토리에서 이웃추가 가능한 계정 추출 (페이지네이션 자동 처리)
4. 각 계정에 서로이웃 신청 — 설정한 최대 수 도달 시 종료

### 기능 2: 서로이웃 댓글

```
로그인 → 서로이웃 목록 수집 → 각 이웃 최신글 방문 → 공감 + AI 댓글
```

1. 서로이웃 관리 페이지에서 그룹 선택 → 정렬 → 이웃 목록 수집
2. 각 이웃의 최신 글 방문 → 공감 클릭
3. 이미 내 댓글이 있으면 스킵
4. AI로 글 맥락에 맞는 댓글 생성 → 등록
5. AI 실패 시 보류 → 마지막에 재시도

### 기능 3: 대댓글

```
로그인 → 내 글 수집 → 각 글의 댓글 탐색 → AI 대댓글
```

1. 내 블로그 PostList에서 기준 날짜 이후 글 수집 (페이지네이션 자동 처리)
2. 각 글 방문 → 댓글 로드 → 다중 페이지 처리
3. 이미 답글 남긴 댓글은 스킵
4. AI로 글 + 댓글 내용 분석 → 맥락에 맞는 대댓글 생성 → 등록
5. AI 실패 시 보류 → 마지막에 재시도

## Gemini API 키 발급

1. https://aistudio.google.com/apikey 접속
2. Google 계정 로그인
3. **"Create API Key"** 클릭
4. 발급된 키를 프로그램의 "Gemini API 키" 필드에 입력
